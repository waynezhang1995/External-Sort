# External-Sort

## Java Implementation of External-Sort

## Compile

```
javac *.java
```

## Run

1. Generate input data (3000000 records)

```
java GenerateData
```

2. Sort

```
java SortPost
```

3. Compare with sample solution

```
java SortUnix
```
